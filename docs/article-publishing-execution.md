## 发布执行流程 (2026-03-17 至 2026-03-19)

### Day 1: 知乎首发 (2026-03-17 20:00)

**步骤**:
1. 登录知乎，进入"写文章"
2. 粘贴文章内容（保留 Markdown 格式）
3. 上传封面图 + 架构图
4. 添加话题标签：`#AI` `#开源` `#GitHub` `#Agent`
5. 设置版权声明：CC BY-NC-SA 4.0
6. 发布后 5 分钟内：
   - 分享到朋友圈（版本 1：专业版）
   - 分享到 AI 技术群（技术群版本）
   - 发微博（版本 1）

**跟进**:
- 第 1 小时：回复前 5 条评论
- 第 1 天：回复所有评论，整理 FAQ
- 第 2 天：统计阅读/点赞/收藏数据

### Day 2: 公众号 + 掘金 (2026-03-18)

**公众号 (08:00)**:
1. 登录公众号后台
2. 使用图文编辑器导入文章
3. 调整格式适配手机阅读
4. 添加原文链接（知乎版）
5. 群发给订阅者

**掘金 (10:00)**:
1. 登录掘金，创建文章
2. 粘贴内容，添加标签：`AI` `开源` `GitHub`
3. 添加掘金专属引言
4. 发布后分享到掘金圈子

### Day 3: Medium + Hacker News (2026-03-18 至 2026-03-19)

**Medium (2026-03-18 18:00)**:
1. 翻译文章为英文（或使用 DeepL）
2. 发布到 Medium，添加标签：`#AI` `#OpenSource` `#GitHub`
3. 提交到 Medium 出版物（如：Towards Data Science）

**Hacker News (2026-03-19 11:00 UTC)**:
1. 登录 HN，点击"Submit"
2. 标题：`Auto-Evolve: A Framework for AI Agent Self-Evolution`
3. URL: `https://github.com/cittaverse/auto-evolve`
4. 评论补充：简短项目介绍 + 实战数据
