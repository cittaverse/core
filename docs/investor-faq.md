## 产品与技术

### Q4: 技术壁垒有多高？

**A**: 

**神经符号架构**是核心壁垒：
- 纯 LLM 方案：黑箱、不可解释、临床不接受
- 纯规则方案：僵化、体验差、依从性低
- 我们的混合架构：LLM 语义理解 + 图论可解释计分

**复制难度**: 
- 架构设计：6-12 个月
- 临床验证：2-3 年 RCT
- 中文口语优化：1 年数据积累

---

### Q5: 大模型迭代对你们的影响？

**A**: 

**风险**: 大模型能力提升可能削弱我们的 NLP 优势

**应对**:
1. **垂直数据护城河**: 2000+ 老年口语数据，通用大模型无此训练数据
2. **临床协议壁垒**: CST 协议规则引擎，非纯 LLM 决策
3. **快速跟进策略**: 使用 API 而非自研，保持技术敏捷性

**结论**: 大模型是工具，不是壁垒。我们的壁垒是临床数据 + 领域知识。
