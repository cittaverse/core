### 2. 描述文案 🟡

#### 短描述 (Tagline) - 60 字符内
```
Let AI Agents evolve themselves
```

#### 完整描述 - 300 字左右
```
Auto-Evolve Framework transforms AI agents from "tools that execute commands" into "partners that evolve together."

The Problem:
Most AI agents today are static - they do what they're told but never improve. Each iteration requires manual intervention, human analysis, and deliberate redesign.

The Solution:
Auto-Evolve implements a closed-loop iteration system: research → execute → verify → learn. Agents can:
- Autonomously discover optimization opportunities based on domain characteristics
- Learn from historical executions to select optimal strategies
- Track quantitative metrics across iterations
- Transparently document all changes and results

GEO Project Validation:
- 4 iterations in 5 days
- 12 GitHub commits across 3 repositories
- ~100,000 words of documentation
- 100% iteration success rate
- Public metrics dashboard for full transparency

Technical Stack:
- Python-based framework
- GitHub Actions for automation
- SQLite for metrics tracking
- MIT License - fully open source

Perfect for:
- GitHub repository optimization
- Content SEO improvement
- Code quality enhancement
- Any iterative development workflow

Try it out, star the repo, and let us know how your agents evolve!
```
