# CittaVerse Product Roadmap Summary

**Updated**: 2026-03-22  
**Full version**: See `auto-evolve` repo → `research/product-roadmap-mvp-to-v2.md`

---

## Three Phases

### MVP (Current → Q2 2026)
**Goal**: First complete user cycle — voice input → narrative scoring → feedback report

- Narrative Scorer v0.5 ✅ (rule-based, 6 dimensions)
- arXiv paper draft ✅ (pending submission)
- Screening questionnaire ✅ (pending deployment)
- ASR integration ⏳ (blocked on API key)
- Minimum UI ⏳ (WeChat H5 or manual flow)

### v1.0 (Q3 2026)
**Goal**: Reproducible service process for 10-50 users

- LLM-enhanced scoring (DashScope/GPT-4o arbitration)
- Multi-dialect ASR (Mandarin + Cantonese + Wu)
- CittaVoice WeChat Mini Program v0.1
- Biographer SOP manual v1.0
- Pricing validation (¥2,999 baseline)

### v2.0 (Q1 2027+)
**Goal**: Scalable platform with biographer ecosystem

- Multi-modal scoring (voice emotion + facial expression + text)
- Independent iOS app + WeChat Mini Program
- Biographer SaaS management platform
- CittaPress auto-typesetting + print integration
- Academic publication (JMIR Aging / CHI)

---

## Competitive Positioning

| Dimension | CittaVerse | Nearest Competitor |
|-----------|------------|-------------------|
| Language | Chinese-optimized | English-first (StoryFile, MemoryLane) |
| Clinical validation | Pilot RCT underway | No clinical studies (most competitors) |
| Scoring framework | 6-dimension automated | Manual or none |
| Open source | MIT license | Proprietary (all competitors) |
| Target | MCI early intervention | General reminiscence or late-stage dementia |

**Market gap**: No product combines clinical validation + automated narrative scoring + Chinese language optimization.
