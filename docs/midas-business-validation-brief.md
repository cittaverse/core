- 中文叙事自动评分（6 维度，开源，无竞品）

**当前状态**：

- ⏳ Pilot RCT 招募中（N=50 目标）

## 4. 三条变现路径（待 Midas 评估）

### 路径 A: B2C 直销 — "人生故事书"
- **客户**：60-80 岁老年人的子女（35-50 岁，付费决策者）
- **产品**：口述采集 → AI 评分 → 专业编辑 → 精装实体书
- **定价假设**：基础版 2999 RMB / 豪华版 5999 RMB
- **获客渠道**：微信视频号 + 社区活动 + 口碑
- **毛利预估**：~60%（主要成本：印刷 + 编辑人工）

### 路径 B: B2B2C 传记师平台
- **客户**：自由传记师、养老社工、心理咨询师
- **产品**：SaaS 工具（AI 辅助采访 + 评分 + 排版）
- **定价假设**：99 RMB/月/传记师 + 每本书抽成 30%
- **规模化优势**：传记师做获客 + 服务，CittaVerse 做技术 + 平台
- **前置条件**：需要 5-10 个种子传记师验证

### 路径 C: B2B 养老机构
- **客户**：养老院、社区日间照护中心、认知康复机构
- **产品**：API 接入 + 定制报告 + 认知追踪
- **定价假设**：年费 5-20 万 RMB/机构
- **壁垒**：需要临床验证数据（Pilot RCT 结果）
- **时间线**：v1.0 之后（2026 Q3 最早）
