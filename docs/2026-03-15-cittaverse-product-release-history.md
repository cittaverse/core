## 2. SEO/GEO 多 Agent 自动化 Loop

> **产品定位**：GitHub 仓库可见性自动化提升系统  
> **目标用户**：开源项目维护者、SEO 团队、开发者关系团队  
> **核心功能**：
> - **GEO 迭代**：每日 4 次自动优化 GitHub 仓库（SEO/外部链接/引用建设）
> - **学术扫描**：每 3 天自动扫描最新学术论文并引用

## 3. L0 质检系统（Pipeline）

> **产品定位**：老年认知障碍 AI 评估引擎（神经符号架构）  
> **目标用户**：养老机构、医院认知科、临床试验团队  
> **核心功能**：
> - **L0 六维认知评估**：感官/情境/情感/连贯性等 6 维度自动评分
> - **元记忆系统**：4 类策略选择器 + Prompt 动态生成，提升评估一致性至 85%
> - **ASR 评估对比**：支持 DashScope/Azure/讯飞多 ASR 引擎对比测试
> - **评委 Agent 并行评分**：4 个评委 Agent 并行 + 1 个仲裁 Agent 处理分歧
> **技术栈**：DashScope（qwen-turbo）+ Python + NetworkX
