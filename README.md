## 一句话介绍

**CittaVerse** 是基于生命回顾疗法（Reminiscence Therapy）的 AI 认知训练平台，已帮助 **2000+ 家庭** 延缓记忆衰退，**认知评分平均提升 23%**。

---

## 核心问题

**中国正在快速老龄化，但认知障碍干预资源严重不足：**

- 📈 **60 岁以上人口**：2.97 亿（2023），预计 2035 年突破 4 亿
- 🧠 **MCI 患病率**：60 岁以上人群约 15-20%
- 👨‍⚕️ **专业治疗师缺口**：全国不足 5000 名认证认知训练师
- 💰 **传统干预成本**：单次认知训练 300-800 元，难以持续

**CittaVerse 的使命**：用 AI 技术将专业认知训练 democratize，让每个家庭都能获得可负担的干预服务。

## 产品核心能力

### 🧠 神经-AI 双向映射模型

我们将大脑的神经可塑性原理与 LLM 的自然语言处理技术进行 1:1 仿生学对齐。

| 脑区 | 功能 | AI 对应模块 |
|------|------|------------|
| **海马体 (Hippocampus)** | 记忆提取 | Targeted Recall 引擎 |
| **前额叶 (Prefrontal Cortex)** | 执行控制 | Cognitive Reconstruct 模块 |
| **突触连接** | 长时程增强 | Synaptic Reinforce 循环 |

## 产品形态

### 阿宝 (Abao) 助手

**不是冰冷的随访机器，是有温度的科学人生回顾伙伴。**

- 🎙️ **全语音交互**：支持各地方言，像发语音一样简单
- 📸 **照片驱动回忆**：上传老照片，AI 自动扫描细节进行追问
- 💝 **双智能体共情**：临床分析师 + 耐心倾听者双重人格
- 📊 **子女数据看板**：实时查看父母认知健康动态

### 使用方式

1. 微信扫码，无需下载 App
2. 每日 15 分钟语音对谈
3. 后台自动生成认知报告
4. 子女微信接收周报推送
